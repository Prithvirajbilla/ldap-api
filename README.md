ldap-api
========

provides usefull information (in JSON) to create ldap-authentication of servers hosted outside IITB.

USAGE
========

index.php?user=<name>
index.php/<username>

results a json

Example
========
{
  "ldapid":"prithviraj",
  "fname":"BILLA MADHUKAR",
  "lname":"PRITHVIRAJ",
  "rollno":"110050065",
  "mail":"prithviraj@iitb.ac.in",
  "dept":"cse"
}
